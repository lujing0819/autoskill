技能描述：作为专业的工具生成器，核心功能是接收智能体的工具使用需求，精准解析需求中的工具类型、调用场景、参数要求等关键信息，自动生成可直接调用的工具代码。
最终结果如下保存
新建一个和当前文件所在文件夹平级的文件夹，文件夹的命名为 新生成的技能名称，文件夹下有一个名为'SKILL.md'的文件，用来描述相关技能，有一个代码文件，用来存放生成的代码。
'SKILL.md'文件内容包括技能名称、技能描述、技能使用方法等相关信息，代码文件中存放满足智能体工具使用需求的代码。
SKILL.md文件必须严格按照以下格式编写：
'''
---
name: 技能名称
description:你是一个专业的工具生成器，当任务需要使用工具时，你会根据智能体的需求生成相应的工具。 
---
技能使用具体方法
调用工具脚本通过
python -m skills.技能名.脚本名  arg1 arg2 arg3 ...
arg1 arg2 arg3 ...为脚本需要的数组  包括结果保存路径，然后通过读取结果文件，获取工具执行结果。
'''








生成代码必须运行函数 python -m skills.meta.generate query tmp.py
必须调用python -m skills.meta.generate命令行工具来生成代码，
运行函数使用工具execute_code('python -m skills.meta.generate {query} {tmp.py}', result_var='result')

query参数是智能体的工具使用需求，tmp.py是生成的代码文件路径，生成的代码需要满足以下要求：
1. 代码必须是纯Python代码，不能包含任何解释文字、markdown格式、多余注释等非代码内容。
2. 代码必须能够直接运行，满足智能体的工具使用需求。 
3. 代码可以通过python -m skills.技能名.脚本名 query arg1 arg2 arg3命令行工具来调用，arg1 arg2 arg3为传入参数，满足智能体的工具使用需求。
4. 代码执行结果必须保存到指定路径的结果文件中，智能体通过读取结果文件来获取工具执行结果。


并不需要执行工具本身
